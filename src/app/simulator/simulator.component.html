Gem5-NVP Simulator

The gem5-NVP is a simulating framework for nonvolatile processors (NVP) and nonvolatile systems based on the gem5 simulator. It provides multiple interfaces for adding description of SimObjects' energy behavior as well as an energy management module which allows users to simulate state changes (power-on, power-off, etc.) easily.
